* Covers:

  * Loss functions and optimizers
  * Training loop basics

### ⚖️ Binary Classification

* Introduces classification problems with two classes
* Covers:

  * Sigmoid activation
  * Binary cross-entropy loss
  * Decision boundaries
  * Model evaluation

---

### 🔢 Multiclass Classification

* Expands to handling multiple classes
* Covers:

  * Softmax activation
  * Cross-entropy loss
  * Accuracy metrics
  * More complex training workflows
